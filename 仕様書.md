# CodecSim 仕様書

> 製品名: **CodecSim**

## 1. 製品概要

### 1.1 コンセプト

音声をリアルタイムでロッシー圧縮コーデック（MP3、Opus、Vorbis、Speex）でエンコード→即座にデコードして出力することで、圧縮後の音質を事前確認できるVSTプラグイン/スタンドアロンアプリケーション。

### 1.2 対象ユーザー

- 音楽制作者（マスタリング時の圧縮音質確認）
- ポッドキャスト制作者
- ゲームオーディオデザイナー
- 配信者（ストリーミング品質の確認）

## 3. 対応コーデック

### 3.1 コーデック一覧

## 4. 実装方法

### 処理の流れ

プログラムがffmpeg.exe(windowsの場合)をコマンドラインから呼び出す。パイプ入力（標準入力）経由で信号を渡し、パイプ出力経由で受け取る。

### 4.2 依存ライブラリ

- **フレームワーク**: iPlug2 (https://github.com/iPlug2/iPlug2OOS)
  D:\programming\booth\codec-simulatorに頓挫したビルド可能なプロジェクトファイルがあるので構造と文法だけは参考になりそう。

## 5 UI

コーデック、ビットレート、内部サンプリングレートを指定できるようにすること。
